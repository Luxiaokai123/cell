# 细胞计数系统功能扩展方案

## 一、概述

本文档描述细胞计数系统的三个功能扩展方案：**参数调节**、**结果导出**、**批量处理**。这些功能将增强系统的实用性和论文展示价值。

---

## 二、参数调节功能

### 2.1 功能描述

允许用户在推理前或推理后动态调整模型参数，实时观察检测结果的变化。

### 2.2 可调节参数

| 参数 | 说明 | 默认值 | 范围 |
|------|------|--------|------|
| **置信度阈值 (conf)** | 过滤低于该阈值的检测框 | RDHS-YOLO: 0.2, DAS-DETR: 0.5 | 0.1 - 0.9 |
| **IOU阈值** | NMS 非极大值抑制阈值 | 0.7 | 0.3 - 0.9 |

### 2.3 界面设计

```
┌─────────────────────────────────────────────────────┐
│  🔧 参数调节                                          │
├─────────────────────────────────────────────────────┤
│                                                      │
│  置信度阈值:  [========●=========] 0.45              │
│              0.1                    0.9              │
│                                                      │
│  IOU阈值:    [=============●=====] 0.70              │
│              0.3                    0.9              │
│                                                      │
│  [应用参数并重新推理]                                  │
│                                                      │
└─────────────────────────────────────────────────────┘
```

## 三、结果导出功能

### 3.1 功能描述

支持将检测结果导出为 Excel 或 CSV 文件，便于后续分析和存档。

### 3.2 导出内容

| 字段 | 说明 |
|------|------|
| 序号 | 检测框编号 |
| 细胞类型 | cell / 血小板 / 白细胞 等 |
| 置信度 | 检测置信度 (0-1) |
| X1, Y1 | 左上角坐标 |
| X2, Y2 | 右下角坐标 |
| 宽度 | 检测框宽度 |
| 高度 | 检测框高度 |
| 面积 | 检测框面积 |

### 3.3 界面设计

```
┌─────────────────────────────────────────────────────┐
│  📊 统计结果                        [导出Excel] [导出CSV] │
├─────────────────────────────────────────────────────┤
│  ...                                                 │
└─────────────────────────────────────────────────────┘
```

### 3.4 技术实现

#### 后端新增 API

```python
from fastapi.responses import StreamingResponse
import io
import pandas as pd

@router.post("/export")
async def export_results(request: ExportRequest):
    """导出检测结果"""
    # 构建 DataFrame
    data = []
    for i, box in enumerate(request.boxes):
        data.append({
            '序号': i + 1,
            '细胞类型': box.class_name,
            '置信度': f"{box.confidence:.4f}",
            'X1': box.x1,
            'Y1': box.y1,
            'X2': box.x2,
            'Y2': box.y2,
            '宽度': box.x2 - box.x1,
            '高度': box.y2 - box.y1,
            '面积': (box.x2 - box.x1) * (box.y2 - box.y1)
        })
    
    df = pd.DataFrame(data)
    
    # 导出为 Excel
    if request.format == 'excel':
        output = io.BytesIO()
        with pd.ExcelWriter(output, engine='openpyxl') as writer:
            df.to_excel(writer, index=False, sheet_name='检测结果')
        output.seek(0)
        return StreamingResponse(
            output,
            media_type='application/vnd.openxmlformats-officedocument.spreadsheetml.sheet',
            headers={'Content-Disposition': f'attachment; filename={request.filename}.xlsx'}
        )
    
    # 导出为 CSV
    else:
        output = io.StringIO()
        df.to_csv(output, index=False, encoding='utf-8-sig')
        output.seek(0)
        return StreamingResponse(
            io.BytesIO(output.getvalue().encode('utf-8-sig')),
            media_type='text/csv',
            headers={'Content-Disposition': f'attachment; filename={request.filename}.csv'}
        )
```

#### 前端调用

### 4.3 技术实现

#### 后端新增 API

```python
class BatchInferenceRequest(BaseModel):
    """批量推理请求"""
    file_ids: List[str]
    model: str
    conf: float = 0.25
    iou: float = 0.7

class BatchResult(BaseModel):
    """单个文件结果"""
    file_id: str
    file_name: str
    model_used: str
    cell_count: int
    processing_time: float
    status: str  # success, failed
    error: Optional[str] = None

class BatchInferenceResponse(BaseModel):
    """批量推理响应"""
    total: int
    completed: int
    failed: int
    results: List[BatchResult]
    summary: dict

@router.post("/batch/inference", response_model=BatchInferenceResponse)
async def batch_inference(request: BatchInferenceRequest):
    """批量推理接口"""
    results = []
    total_cells = 0
    total_time = 0
    
    for file_id in request.file_ids:
        start_time = time.time()
        try:
            # 调用单图推理
            model, config, error = get_model(request.model, base_dir)
            if model:
                # 执行推理...
                result = model(file_path, conf=request.conf, iou=request.iou)
                cell_count = len(result[0].boxes)
                total_cells += cell_count
                proc_time = time.time() - start_time
                total_time += proc_time
                
                results.append(BatchResult(
                    file_id=file_id,
                    file_name=file_id,
                    model_used=request.model,
                    cell_count=cell_count,
                    processing_time=proc_time,
                    status="success"
                ))
        except Exception as e:
            results.append(BatchResult(
                file_id=file_id,
                file_name=file_id,
                model_used=request.model,
                cell_count=0,
                processing_time=0,
                status="failed",
                error=str(e)
            ))
    
    return BatchInferenceResponse(
        total=len(request.file_ids),
        completed=sum(1 for r in results if r.status == "success"),
        failed=sum(1 for r in results if r.status == "failed"),
        results=results,
        summary={
            "total_cells": total_cells,
            "avg_cells_per_image": total_cells / len(results) if results else 0,
            "avg_processing_time": total_time / len(results) if results else 0,
            "total_processing_time": total_time
        }
    )
```
